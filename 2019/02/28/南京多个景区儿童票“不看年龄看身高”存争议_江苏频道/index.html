南京多个景区儿童票“不看年龄看身高”存争议_江苏频道

发表于 2019-02-28 | 浏览量

近日，广东消费者委员会就广州长隆集团多个场所存在以身高作为未成年人优惠票标准的问题，代表消费者向广州市中级人民法院提起消费民事公益诉讼。广州市中级人民法院已受理此案。据了解，这是全国第一宗未成年人消费权益保护公益诉讼。

旅游景区、公共交通等领域的儿童票该按照年龄还是身高为标准，一直存在较大的争议。
2018年，广东的刘先生因10岁女儿身高超过1.4米被上海迪士尼要求购买成人票，向上海法院提起民事诉讼。现代快报记者曾关注报道此事，不过该案尚在审理中，目前还未有宣判信息。
那么南京的景区儿童免票标准是什么呢？“儿童票”究竟该以什么为准？2月26日，现代快报记者调查了南京12家适合未成年游览的收费景区发现，其中南京海底世界、红山森林动物园等6家景区完全以“看身高”为主，明孝陵景区、中国科举博物馆按照身高、年龄“双标准”免票，总统府、六朝博物馆等4家景区是为数不多的以年龄条件来免票的景区，且免票范围基本上覆盖到了所有的未成年人。
南京一家收费景区工作人员告诉记者,由于标准不一，具体到售票和检票环节，景区一般采用“目测大法”，以“看身高”“看脸”为主，“一般如果不是超出太多,‘睁一只眼闭一只眼’就过去了。”
不光景区的儿童票标准五花八门，现代快报记者还发现,同样是交通领域, 陆运、水运和空运不一样、“大交通”和“小交通”也不同。以南京公交为例，目前，儿童免费乘坐公交的身高标准是1.3米。而中国铁路客户服务中心官网显示，每一成人旅客可免费携带一名身高不足1.2米的儿童。
早在2012年，国家发改委发布的《关于进一步落实青少年门票价格优惠政策的通知》就提出了青少年门票价格政策：对6周岁(含6周岁)以下或身高1.2米（含1.2米）以下的儿童实行免票；对6周岁（不含6周岁）～18周岁（含18周岁）未成年人实行半票。
但具体到不同的省份，标准并不一样，一些省份给出了更加宽松的“富余”。比如，2018年1月，江苏省物价局印发的《江苏省景区门票和相关服务价格管理办法》中，进一步明确了对6周岁（含6周岁）以下或者身高1.4米（含1.4米）以下的儿童凭有效身份证件免收门票的规定。不过，这一规定，主要是针对实行政府指导价的景区。
业内人士表示，江苏“双标准”免票政策能够覆盖绝大多数学龄前儿童，而像南京海底世界、常州恐龙园这样的社会民营景点，门票实行的是市场调节价，政府只能鼓励建议其制定合理的儿童票价。但从记者调查的南京12家收费景区来看，大都没有实行6周岁或1.4米免票的双标准。
南京市民李女士的孩子今年9岁，身高已接近1.5米。李女士希望景点能根据年龄制定儿童票的售票标准。
“我觉得按年龄更公平，‘儿童票’是属于儿童的福利，应该让孩子都能平等享受得到。”市民刘女士也很赞成儿童票收取以年龄为准。在她看来，儿童发育有快有慢，现在孩子营养都比较好，按身高免票未免太“过时”，有的幼儿园孩子身高就已超过了免票线。
一位网友表示，景区门票这一块，应该以年龄为准。但交通工具的儿童票购买标准应该以是否单独占座为衡量标准，毕竟是占用公共交通的资源。
去年4月，中国青年报社会调查中心联合问卷网，对1969名儿童家长进行的一项调查显示，56.7%的受访家长认为以身高作为儿童票收取标准不合理，67.1%的受访家长赞同儿童票收取以年龄为准。
