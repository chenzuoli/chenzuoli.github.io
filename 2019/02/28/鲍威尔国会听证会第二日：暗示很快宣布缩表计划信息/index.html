鲍威尔国会听证会第二日：暗示很快宣布缩表计划信息

发表于 2019-02-28 | 浏览量

美联储主席鲍威尔的国会证词陈述进入第二日。周三北京时间27日晚23:00，鲍威尔向众议院金融服务委员会就美国货币政策和经济情况进行半年一度的证词陈述。



后歆桐
美联储主席鲍威尔的国会证词陈述进入第二日。周三北京时间27日晚23:00，鲍威尔向众议院金融服务委员会就美国货币政策和经济情况进行半年一度的证词陈述。
鲍威尔在证词陈述中重申了美国政府债务的不可持续性，并暗示将“很快”宣布缩表计划相关信息。
在今日的证词陈述中，对于美国经济的情况和前景，鲍威尔表示，2018年美国GDP增速似乎接近3%，2019年可能略微低于该水平；财政计划对2018年需求构成支撑；最近一年，美国薪资一直在上扬，很难判断监管改革是否推动美国经济增长。他并称，美联储并没有寻求更高的通胀目标，正在寻求用更好的方式达成通胀目标。
鲍威尔还再度提及美国政府债务问题。他指出，美国政府的支出方式不可持续是长期性问题。由于联邦政府债务比国内GDP增长还快，所以形成联邦政府预算的不可持续性。
此外，他表示，美国家庭债务整体基本健康，学生债务是担忧的一个问题，贷款抵押债券（CLO）构成宏观经济风险，但并未构成系统性风险。他还指出，杠杆贷款是一个重要的监管点，但未对美国银行业构成系统性风险。
在昨日的参议院听证会上，鲍威尔也回答了关于债务问题及“绿色新政”的问题。
参议员David Perdue向鲍威尔询问了对“现代货币理论（MMT）”的看法。MMT背后的逻辑是只要美联储能够在不引发通胀的情况下保持低利率，国家债务问题和预算赤字就不是问题。MMT倡导者认为美联储将成为通过宽松货币政策为这些计划提供资金的合作伙伴。
对此，鲍威尔表示
赤字对那些能够以本国货币进行借贷的国家来说不重要的想法是错误的。他认为，相对国内生产总值，美国的债务处于相当高的水平，更重要的是，其增速超过了国内生产总值的增长。
而在回答参议员Jon Tester提出的关于提高债务上限的问题是，鲍威尔称，“这甚至超出了考虑范围。美国不履行义务，偿还到期债务的想法根本不应该被考虑。”
他补充道，他将美国政府违约的前景视为“一条醒目的线，我希望我们永远不会越过它”。
此外，鲍威尔昨日还回答了美联储资产负债表是否有助于为经济和环境项目的“绿色新政”（Green New Deal）提供资金的问题。
“绿色新政”是由新当选的纽约州民主党议员Alexandria Ocasio-Cortez和马萨诸塞州民主党参议员Ed Markey最近宣布的一项绿色革命的计划。其提议包括在12年内减少美国的碳排放，启动一项大规模的投资计划，在保证就业保障的同时，为太阳能、风能和其他可再生能源改造旧建筑。
对此，鲍威尔称，“我们的角色不是在‘环境、社会或其他关键问题上’为特定的政策提供支持。”
就在鲍威尔在众议院作证的同时，包括美国1月季调后成屋签约销售指数月率、美国12月耐用品订单月率终值等在内的一系列美国最新经济数据陆续出炉，喜忧参半。
美国12月工厂订单环比0.1%，预期0.5%，前值-0.6%。美国1月成屋签约销售指数环比4.6%，预期0.4%，前值-2.2%。美国12月耐用品订单环比终值1.2%，预期1.2%，前值1.2%。27日晚稍早公布的数据显示，美国去年12月商品贸易帐逆差795亿美元，逆差规模创历史新高。
Columbia Threadneedle Investments的全球固定收益副主管特努佐（Gene Tannuzzo）表示，全球经济正走向衰退，意大利等一些国家已经出现萎缩，工业生产和贸易指标继续下降。如果这些全球压力持续存在，美国将继续出现增长减速。美国2018年的经济起点是相当强劲的，平均增长超过3%，所以现在说美国在今年上半年陷入衰退还为时过早。但他认为如果接近年底，全球大环境不改善，美国经济将面临风险。
在货币政策方面，鲍威尔在证词中再度重申，将保持耐心，并观察经济形势的演变。他还暗示，将“很快”宣布资产负债表计划相关的消息，并表示维持金融稳定是美联储职责的一部分。
此前，克利夫兰联储主席梅斯特（Loretta Mester）也曾透露，美联储将在“即将举行的（3月）会议中”敲定关于结束缩表的计划。
而鲍威尔在周二参议院金融银行委员会作证时表示，不断加剧的风险和近期疲弱的数据不太可能阻止美国经济在今年稳步增长，但美联储在决定进一步加息时仍将保持“耐心”。
鲍威尔连续两日在国会听证会上的言论夯实了美联储上月的政策转变。 自上月初以来，对加息保持“耐心”的立场一直是美联储官员讲话的主要内容。
“只要经济稳步成长，且不引发通胀，那么美联储应该会按兵不动，”Cresset Wealth Advisors首席投资官Jack Ablin表示。
CIBC资本市场经济学家Avery Shenfeld表示，鲍威尔的公开声明与1月的会议纪要区别不大，“美联储展望比较中性，但鲍威尔在强调经济状况和预期展望方面仍有一些“鹰派”口气。这些仍然不能改变我们的看法，那就是上半年美联储将维持利率不变。”
鲍威尔还表示，美联储在制定政策时不会考虑政治因素，股市是美联储监控的众多方面之一。
然而，美股却并不“买账“。不同于周二鲍威尔讲话时的止跌转涨，今日鲍威尔发表证词陈述时，美股继续维持跌势。
近来频频走软的美元指数在鲍威尔进行证词陈述期间继续走低，一度跌破96刷新日低。截至第一财经记者凌晨1点左右截稿时，反弹至96.12。
瑞穗证券首席外汇策略师山本雅文表示，“美元下跌有点让人困惑，因为鲍威尔的言论并无新意。不过，市场对美元的潜在看跌因素极其敏感，英镑走强也有所影响“。
10年期美债收益率在鲍威尔周三讲话期间升破2.68%，刷新日高。昨日的参议院听证会期间，美债收益率也出现反弹。
不过，特努佐称，当美国陷入经济衰退，我们将看到信贷息差扩大公司债券表现不佳以及国债收益率走低。作为对市场的反馈，美联储可能会降息，并可能推出更多的量化宽松政策。一旦达到那个时候，公司债券将开始表现良好，风头将盖过美国国债市场。
黄金期货在鲍威尔证词阐述期间走势波动，一度下跌后出现小幅反弹。
永丰金融集团研究部主管涂国彬称：“金价在1300美元寻获良好支撑，特别是美联储最新立场出炉后。目前人们对于美联储政策前景看法差异甚大。但美联储耐心以对，且更加倚重数据，这将对金价有利。今年合理区间应在1300～1400美元。”

鲍威尔称准备根据经济和金融的发展来调整缩表的任何细节。从长期来看，资产负债表的规模将取决于对美联储负债（如货币和银行储备金）的需求。
美联储1月利率决议及其后鲍威尔的“鸽派”发言后，美股三大股指大幅收高，黄金一度突破1320美元关口，美元跌至4个月低位。
特朗普无心插柳，美国政府部分停摆或影响美联储利率决议。
律师出身的鲍威尔曾被嘲笑为“非典型”总统任命的“非典型”美联储主席。
近期美联储持续释放鸽派信号，美元近一个月下跌2.2%短期或见顶，金价同期则逆势上涨6%。
