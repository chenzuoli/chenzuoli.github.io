故宫“火锅店”成大热门舌尖上的文创之路怎么走？_吉林频道

发表于 2019-02-28 | 浏览量

(记者上官云)故宫又双叒叕刷屏了！前不久是因为“上元之夜”，这次是因为令人食指大动的美食……一些吃货们奔走相告，“故宫开火锅店了！”
先来看这份“圣旨菜单”：“奉天承运，皇帝诏曰，汤底——万寿菊花锅，酱料——江山社稷酱，涮肉——科尔沁牛上脑……”
店内主打“万寿菊花锅”，相传由慈禧发明，也是她相当喜欢的一种火锅吃法。蘸料的灵感来源于江山社稷五色土，将青红黄白黑五种颜色调制五种蘸酱，酸辣鲜香，满足不同口味需求。
人人都说美食美器，坐在这里吃一顿，难怪有网友感叹“热气氤氲中，像是奉旨涮锅”。
自打开业，餐厅基本处于爆满状态。为了品尝“宫味儿”火锅，许多人在下午就在门口排起长队，但店里在晚上5点半后才供应火锅……
离着“火锅店”不远的咖啡厅，“一座难求”的场景也时不时上演。店内陈设沿袭了简单大气的风格，木桌、木椅还有隔扇错落有致，一眼望去，古色古香。
据说，故宫咖啡厅里有30多种饮品可以选择，“养心咖啡”、“佛手香茗”、“一骑红尘妃子笑”荔枝红茶……“千里江山卷”等别出心裁的甜品早早成了爆款。
有人说，站在紫禁城的城墙根儿，捧着“康熙最爱巧克力”，哪怕是只绕着故宫走一圈，也是相当惬意了。
所以，在这些地方看到手捧咖啡杯忙着各种自拍合影的人，千万不要惊讶，毕竟这几处都是新晋的打卡圣地。
还有机智的网友很快根据故宫咖啡厅、“火锅店”所在的位置，迅速划出了最佳参观路线，走过三大殿，陶瓷馆、珍宝馆一个不落，爬城墙、看展览，最后舒舒服服坐下来吃顿饭。当然，得排到位置……
其实，再早一些，故宫“冰窖餐厅”也是人们心中温暖的“食堂”。
里面巨大的冰凌吊灯将整个屋子映衬的朴素雅致，石头墙上搭配琉璃制成的冰块装饰，浓厚的历史气息扑面而来。
据说，这里的菜肴多以套餐形式出现。“皇家藏冰”“御制棒棒糖”……不止一位观众给它点赞，说“创意很新颖”。
在故宫供应火锅、开咖啡厅之外，还出现了其他与饮食有关的文创。比如，中国国家博物馆发布了新的文创产品——棒棒糖，其中一款灵感来自青花瓷，其他三款来自粉彩瓷。
广东省博物馆也不甘落后，推出了几款“蛋糕”文创，包括“粤藏”、 “粤光宝盒”以及“风尚”，制作无一不是花尽心思。
比如，每一层夹心都要使用特定的模具来制作，做好后要冷冻定型，最后在蛋糕外层喷涂，画出广东省博物馆的logo……
故宫则在中秋节时推出“特色月饼”。曾经有人说，没想到吃个月饼还吃出了满满地文化感。
综合上述几家博物馆的做法，有网友评价，这是给博物馆文创提供了“新玩法”和可能性，完全可以叫做“舌尖上的文创”，也让古老的文物真正活了起来。
只不过，这种“新玩法”可能还需要一些时间。
比如，故宫“火锅店”红了以后，争议也随之出现了。不止一位去“拔草”的网友觉得菜品虽挺有特色，但肉类和蔬菜价格偏高，性价比太低。
另外有网友大概是联想到了小品《打工奇遇》，调侃道“宫廷玉液酒，一百八一杯”；或者还有人戏谑着提到一份小料12元，一份肉100元，“假如我是和珅”……
有些网友的评价更是直接嫌弃价格，“吃不起，只好等着别人请了”“看来我可以买份小料，再要一个蔬菜拼盘，就是不知道肉能不能自带……”
当然也有人说，在这里吃饭，从某种角度来说，享受的是其中深厚的文化底蕴，人均200真的还好，并不算太贵。况且淡季门票已经很低，要看到故宫在创新方面的努力。
其他一些人操心的却是文物，提出故宫不宜过度商业化，更觉得火锅可能蒸汽比较大，千万别因此影响了古建筑。
换个视角，包括故宫在内，各种涉及或不涉及餐饮的博物馆文创成为热门，折射的是人们不断提升的文化消费需求。
文创们需要满足的，也正是这种消费需求。
所以，很多人为故宫“火锅店”点赞，觉得这是服务观众的一种好做法，也是观众更好体验故宫文化的一个新渠道。
而且有人认为，故宫角楼应该不是文物收藏密集区，在做好安全防护的前提下，赋予适度的餐饮休闲功能，可以提升故宫的亲和力。故宫也在防火安全方面做了很充足的准备。
“故宫火锅”太过于稀缺。人们愿意预约、排队来故宫吃火锅，追求的也许并不只是舌尖上的醇美味道，而是愿意体验一餐饮食承载的深厚底蕴。
因而，从这个角度来说，在故宫吃火锅的人，在意的可能不会是花了多少钱，而是能否真正身临其境得到文化氛围的浸润。
文创美食更大的价值，大概正在于此。如果一味用金钱来衡量，反而可能不太恰当。当然，火锅也好、咖啡也罢，自然还是越接地气越好。
近年来能够看到，一些博物馆在文创开发上绞尽脑汁，棒棒糖、蛋糕等都是思路新颖，毕竟“民以食为天”，“吃”总是可能受到关注，再有厚重文化的加持，想不红大概都难。让文物“更鲜活”，总是需要一些有益尝试。
但这样的尝试，也许还需要慢慢探索。就像一位网友评价的那样，提供餐饮和单纯的饮食文创还不太一样，故宫IP形成得不容易，需要细节与关怀来呵护。文创做好了，实在是一件锦上添花的事情。(完)
