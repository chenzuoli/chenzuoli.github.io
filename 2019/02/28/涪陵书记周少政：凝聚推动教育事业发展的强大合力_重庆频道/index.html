原标题：
加快推动教育高质量发展 努力办好人民满意的教育
巴渝传媒网讯（记者 杨莉菊）近日，涪陵区委、区政府召开全区教育大会，深入学习贯彻习近平总书记关于教育的重要论述，全面贯彻落实全国、全市教育大会精神，安排部署全区教育工作。区委书记周少政出席并讲话，区委副书记、区政府区长种及灵就全区教育工作进行安排部署。
区人大常委会主任李景耀、区政协主席徐志红等区四大班子有关领导出席会议。会议由区委常委、区政府常务副区长杜海波主持。
周少政指出，习近平总书记就教育改革发展作出一系列重要讲话、重要指示批示，为新时代教育改革发展指明了方向、提供了根本遵循。全区上下要深入学习贯彻总书记关于教育的重要论述，充分认识新时代教育的战略地位和重大意义，深刻把握总书记关于教育事业发展的新理念新思想新观点，推动总书记关于教育的重要论述在涪陵落地落实，努力办好人民满意的教育，推动涪陵教育事业不断迈上新台阶。
周少政要求，全面落实立德树人根本任务，让党和国家事业后继有人，要坚持正确的育人方向、育人目标、育人路径，解决好“为谁培养人”“培养什么人”“怎么培养人”的问题，从铸就坚定理想信念、弘扬爱国主义情怀、强化思想品德养成、不断增长知识见识、培养艰苦奋斗精神、增强综合素质能力方面下功夫。要统筹协调发展，优化教育结构；积极探索创新，凸显教育特色；坚持人民共享、促进教育公平；坚持德能并举，提高教师队伍素质，加快推动教育高质量发展。要深入推进教育领域综合改革，深化评价制度、办学体制、教育管理、保障机制改革，不断激发全区教育事业发展的生机活力。
周少政强调，要切实加强党对教育工作的全面领导，不断凝聚推动全区教育事业发展的强大合力。要压实领导责任，党政主要领导要熟悉教育、关心教育、支持教育、发展教育。要强化党建工作，严格履行管党治党主体责任。要突出思想引领，把思想政治工作贯穿到教育教学工作始终。要统筹调动家庭、学校、政府、社会各方面教育资源，凝聚发展合力。
种及灵指出，要深学笃用习近平总书记关于教育的重要论述，坚持教育优先发展。要着力提升教育供给能力，加快优化教育资源布局，着力夯实义务教育根基，全面保障困难群体受教育权利。要全面提高育人质量，协调发展各类教育、着力提升学生综合素质，积极打造教育品牌，着力提升教育服务高质量发展能力。要切实加强教师队伍建设，强化培养培训，注重关心关爱，严格管理督导，着力提升教育内涵引领能力。要持续深化教育综合改革，全面深化教育“放管服”、教育教学改革，推进办学体制、普通高考综合改革，着力提升教育创新发展能力。要着力提升对教育发展的保障能力，进一步强化政府统筹，加大教育投入，落实管理职责，维护校园安全，进一步为教育发展创造良好环境。
长江师范学院、区教委、涪陵十四中、珍溪镇等4个单位的负责同志在会上作了交流发言。
2月20日，在涪陵区教育大会上，长江师范学院、区教委、涪陵十四中、珍溪镇等4个单位的负责同志作了交流发言。
全区教育大会深入学习贯彻习近平总书记关于教育的重要论述，全面贯彻落实全国、全市教育大会精神，对新时代涪陵教育发展作出战略新部署、规划新设计和工作新安排，在涪陵各项事业发展中具有基础性、全局性、先导性作用，对涪陵区委区政府团结和带领全区人民打好“三大攻坚战”、落实“八项行动计划”、做好“三篇大文章”、推进“两线三片一城”建设具有极其重要的价值和意义。长江师范学院将以此为契机，更加精准落实好陈敏尔书记在全市教育大会上对高等教育提出的“三贴近”要求，全力为涪陵建设重庆市教育强区、区域创新高地和区域中心城市提供有力的人才保障、科技支撑和智力支持。
区委、区政府召开高规格教育大会，确立了教育改革发展新思路、新路径，作出了新安排、新部署，全区教育系统将认真贯彻落实教育大会精神，坚持项目化管理，表格化推进，进一步深化教育领域综合改革。下一步，全区教育系统将聚焦目标定位，开拓教育发展新格局，切实把思想和行动统一到党中央国务院、市委市政府和区委区政府的决策部署上来，把工作和步调落实到全国、全市、全区教育大会的安排要求中去，用使命和担当加快推进教育现代化。通过聚焦立德树人，实现育人质量新突破；聚焦内涵引领，开创师资建设新境界；聚焦改革创新，释放教育发展新活力，用创新和作为加快推进全区教育高质量发展，加快建设重庆市教育强区。
立德树人，教育之根本。在习近平新时代中国特色社会主义思想指引下，形成了新的育人结构，概括为一个核心：价值育人；两翼驱动:教师育人、家长育人；三大抓手: 文化育人、榜样育人、健康育人；四个保障：创新育人、实践育人、公平育人和开放育人。涪陵十四中将坚持立德树人，促进学生全面发展；坚持文化引领，推进学校内涵发展；坚持开放办学，助推涪陵教育优质发展，办好人民满意的学校，奋力书写新时代学校发展的新篇章。
教育是国之大计、党之大计。打造全区教育强镇，是镇党委政府的一致目标。近年来，珍溪镇教育事业发展取得一定的成绩，但以高质量发展的标准来衡量还存在差距。下一步，全镇上下将认真学习习近平总书记关于教育的重要论述，切实提高对教育重要性的认识，坚持优先发展教育事业，确立更高的目标追求，加强领导，营造良好教育教学氛围，抓好改善办学条件、校园安全保障、维护教育公平等重点工作，统筹政府、学校、家庭、社会资源，共同关心，形成合力，以高质量的工作办人民满意的优质教育，努力打造全区教育强镇。
记者 杨莉菊 整理
2月20日，涪陵区教育工作大会的隆重召开，让教育工作者们倍感振奋，大家纷纷表示，将深入学习习近平总书记关于教育工作的重要论述，深入贯彻落实全国、全市、全区教育大会精神，积极作为，为加快推进全区教育高质量发展、加快建设重庆市教育强区贡献力量。
全区教育工作大会对全区教育工作作了全面部署，让我们深受鼓舞。会议指出，加快推进全区教育高质量发展，推进义务教育优质均衡发展，让人很受启发。作为一名乡村基层教育工作者，要聚焦乡村教育振兴，迅速行动，勇敢实践。
下一步，我们将在本次大会精神的指引下，积极投身于涪陵区教育综合改革，落实好合作办学举措，开展好乡村学校与城区优质学校合作办学。实施好乡镇区域内校长教师交流轮岗，优化乡村学校师资，努力培养德、智、体、美、劳“五育”并举、全面发展的学生；建设好每一所乡村学校，配齐配足乡村学校教学、生活设备设施，完善寄宿制学校办学功能，努力建设好老百姓身边的乡村学校。认真落实好乡村学校教育教学管理，做实教育活动过程，努力培育热爱学习、兴趣广泛，热爱祖国、忠诚于党的学生。
全区教育大会传达贯彻了全国、全市教育大会精神，对涪陵区新时代教育的发展既发出了动员令，又画出了路线图，还制定了时间表。大会明确提出建设重庆市教育强区的奋斗目标，让我们倍感振奋。特别是会议提出实施普通高中提质攻坚计划、打造重庆市高中教育品牌基地，让我们信心百倍。
接下来，我们的工作重点是：学习贯彻全区教育大会精神，用新思想激发新作为。丰富学生活动，创新德育途径，将立德树人的根本任务落到实处。提升教师队伍业务水平，强化课程建设，优化课堂教学，让学生考出好成绩，考上好大学。有序推进高考综合改革的实施。对当前选课走班模式进行精心策划；对学生职业生涯规划进行精心指导；为创新、拔尖人才的培养搭建更多平台，以适应高校自主招生的需要。
教育是民族振兴、社会进步的基石。我们将认真贯彻全区教育大会精神，践行教育工作者的职业信念和价值追求，不断提高教学质量，为加快推进教育现代化、建设重庆市教育强区添砖加瓦。
接下来，我们要认真做好学校优质均衡发展、划片招生、均衡分班、搭配教师、开齐开足课程、多维度统一考核考评，深化教育改革，全面提升学生的综合素质和能力。推进区域教育优质均衡发展，积极参与全区合并、合作、托管办学模式改革，努力提高办学水平和教学质量，为创建国家义务教育优质均衡区作出贡献。
