原标题：从严从实抓好中央巡视反馈问题整改确保如期高质量打赢精准脱贫攻坚战
根据中央第十一巡视组脱贫攻坚专项巡视反馈意见，2月27日，省委常委会利用1天时间召开脱贫攻坚专项巡视整改专题民主生活会。会议坚持以习近平新时代中国特色社会主义思想为指导，深入学习贯彻习近平总书记关于扶贫工作、巡视工作的重要论述、重要指示批示和视察安徽重要讲话精神，从树牢“四个意识”、坚定“四个自信”、做到“两个维护”出发，聚焦中央专项巡视反馈意见，在深入调研的基础上，对照问题找差距，剖析思想查原因，严肃开展批评和自我批评。
省委书记李锦斌主持会议并作总结讲话。省委副书记、省长李国英，省委副书记信长星，省委常委出席会议。中央纪委国家监委、中央组织部有关同志到会指导。
省委常委会对开好这次专题民主生活会高度重视，扎实做好会前准备工作。突出抓学习，全面提高思想认识，打牢了开好会议的思想基础；突出抓调研，全面找准查实问题，强化了开好会议的实践基础；突出抓整改，全面推进任务落实，成立高规格整改领导小组，细化分解中央专项巡视反馈问题整改，夯实了开好会议的组织基础；突出抓检查，全面深刻对照反思，认真撰写对照检查材料，把牢了开好会议的行动基础。
会上，李锦斌首先代表省委常委会作对照检查，根据中央专项巡视反馈意见，从5个方面查找问题并进行了深刻原因剖析，提出了6个方面、19条具体整改措施。紧接着，李锦斌作个人对照检查，省委常委依次进行。每位常委对照检查后，其他常委立即对其开展批评。自我批评时，大家刀刃向内、自我亮剑；相互批评时，大家开门见山、一针见血；听取批评意见时，大家虚心接受、照单全收，充分体现了自我净化、自我完善、自我革新、自我提高的政治自觉、思想自觉和行动自觉。
李锦斌在总结讲话中说，这次专题民主生活会贯彻认真态度和整风精神，“病根”找得准，“病症”点得实，“病灶”挖得深，开出了高质量、高水平，是一次再学习、再体检、再提高的会议。要围绕做好巡视“后半篇文章”，在抓整改落实上见真章、动真格、求实效，坚决做到“四确保四不放过”。要紧盯要求抓紧改，没有出台整改方案的要抓紧制定，已经出台的要抓紧实施，确保按时限要求高标准高质量完成。要同频共振一体改，各级各有关部门既要各司其职、各负其责，又要注重衔接、协调配合，做到步调一致、上下联动、一体解决。要标本兼治彻底改，一手抓问题整改，一手抓制度建设，努力从源头上铲除问题滋生土壤。
李锦斌指出，今年是习近平总书记发表“三严三实”重要论述5周年，要坚定不移把“三严三实”要求贯穿巡视整改和扶贫开发的全过程和各方面，确保巡视反馈问题改彻底改到位，确保以习近平同志为核心的党中央脱贫攻坚决策部署在安徽落地生根，确保如期高质量打赢精准脱贫攻坚战。一是政治上要“高站位”，从严从实抓好习近平总书记关于扶贫工作重要论述的学习贯彻。要在学习中吃透精髓要义，切实把习近平总书记重要论述内化为推进精准脱贫的本领、外化为高质量脱贫的实效；在学习中把准方针政策，把学习贯彻习近平总书记重要论述的过程变成精准理解政策、把握政策、执行政策的过程；在学习中明晰目标标准，不折不扣执行“两不愁三保障”标准。二是工作上要“重摆位”，从严从实推进大别山等革命老区脱贫攻坚。感情要再倾注，全力做好帮扶工作；政策要再倾斜，全面抓好支持大别山等革命老区脱贫攻坚《实施意见》的贯彻落实；扶志要再倾力，大力弘扬革命老区精神，引导贫困群众依靠顽强意志和勤劳双手脱贫致富。三是战役上要“攻点位”，从严从实落实精准扶贫精准脱贫基本方略。要把好精准识别“源头关”，对深度贫困县认定标准和程序等情况进行“回头看”，提高识别精准度；锁定精准施策“靶向点”，进一步完善县级脱贫攻坚项目库建设管理，强化资金监管，抓好“四送四通”直通车扶贫举措；落实精准帮扶“硬杠杠”，持续强化“四带一自”产业扶贫等帮扶措施，切实提高脱贫质量；筑牢精准脱贫“稳定器”，健全完善贫困退出机制，健全稳定脱贫长效机制。四是作风上要“转到位”，从严从实整治形式主义官僚主义。要坚持从省委常委会做起，从省委常委做起，认识上要更高一些，带头把自己摆进去，把工作摆进去，把职责摆进去；行动上要更快一些，带头践行“十戒十做”，带头开展“学查改立”；工作上要更实一些，注重从具体事抓起，从身边事做起，从群众最不满意的地方改起，真正做到打铁自身硬、正人先正己。
李锦斌强调，抓好巡视整改、打赢脱贫攻坚战，关键要砥砺担当精神，压紧压实责任。要牵住主体责任“牛鼻子”，坚持省负总责，直接抓、抓具体、抓到位。要种好分管领域“责任田”，常委同志要按照职责分工坚决扛起分管责任，督促指导分管领域和部门全力做好巡视整改和脱贫攻坚工作。要架起监督执纪“探照灯”，省纪委监委要严肃查处脱贫攻坚中的违纪违法行为，扎实推进监察体制改革向基层、向一线、向群众身边延伸，让人民群众切身感受到巡视整改和反腐倡廉带来的新变化新气象，以优异成绩庆祝新中国成立70周年。（通讯员 宗禾）
