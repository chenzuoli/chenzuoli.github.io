北京时间2月25日上午（当地时间2月24日晚），第91届奥斯卡金像奖颁奖礼在美国洛杉矶举行，由华裔女导演石之予执导的8分钟动画短片《包宝宝》，喜获最佳动画短片奖。大家为石之予庆贺骄傲的同时，其实还不知道本片中还有一位华裔女孩的身影——她就是皮克斯艺术总监、本片美术总监刘碧青（英文名Lona Liu），一个地道的南昌姑娘。
刘碧青1987年出生在南昌，在豫章小学读到四年级，后来跟随妈妈到美国生活，毕业于全美设计院校中排前五名的明星学校——巴沙狄那艺术中心设计学院。除了《包宝宝》，她还担纲过《恐龙当家》《海底总动员2》《寻梦环游记》《超能总动员2》等著名动画影片的制作。为此，本报第一时间采访了她的父亲刘溯并联系上在美国的刘碧青进行独家专访。
《包宝宝》获得奥斯卡最佳动画短片奖，刘碧青的家人也沉浸在获奖的喜悦当中，特别是父亲刘溯，抑制不住从心底里透出自豪。对于女儿的成长和与故乡的联系，刘溯讲得很仔细。
十几岁的时候，刘溯和家人带刘碧青去看《海底总动员》，出了电影院，她一直低着头闷着不说话，突然间蹦出来一句：“我长大了要去迪斯尼画动画，有一天我也要画出大家喜欢的动画电影出来。”
机遇在不经意间走到身边。刘溯现在还清楚地记得，几年前的一天，女儿高兴地告诉他，有一个华裔女孩（石之予）和她准备拍一部关于“中国包子”的故事，并嘱咐他保密，后来影片公布他才知道这是《包宝宝》。
这两个来自中国的女孩有着差不多的成长背景和童年记忆：妈妈包的饺子、包子，深藏着无尽的母爱，变成她们长大后心中最温馨的记忆。类似的生活经历，让两个来自不同地方却有着相同血脉传承的东方女孩有了强烈的共鸣，于是，她们联手开始创作了这部以美食来表达亲情的动画短片——《包宝宝》，她们希望用这部短片来表达对母亲由衷的感激。两个年轻女孩的创意打动了公司的高层，她们在高手如林的皮克斯脱颖而出，打造出了这部充满着中国人情味的动画短片，两个来自中国的女孩也因此创造了奇迹。
25日奥斯卡奖项揭奖，刘碧青当即就给在国内的父亲发了短信。记得在影院，刘溯一眼就看到在影片中，女儿偷偷把父母亲当年的照片放进了背景里，那是她儿时的记忆，作为父亲的他一下子就眼眶湿润了。小学四年级才离开南昌的刘碧青对亲情很看重。她每次回国都会回南昌看爷爷奶奶，特别是做了《寻梦环游记》和《包宝宝》后，更是对老人呵护备至。
她喜欢吃南昌的猪血粉，喜欢奶奶做的“扣肉”。每次她回国，家人都会帮她炒一碗米粉。南昌的大街小巷都有着她的回忆，她都会去走走看看。
刘碧青希望观众看《包宝宝》这部动画片时，能和她一样感受到在妈妈家里的那种温暖、安全和爱。
“在我和女儿的情感中，南昌始终是我们割舍不掉的情感地。”谈起家乡南昌，刘溯打开了话匣子。
当年刘溯夫妻都是在外地工作，母亲在怀碧青的时候从四川调回南方工作，于是她就在南昌出生了。生下没多久，母亲就去新单位报到，丢下两个月的她给爷爷奶奶看护，这一看护就是好多年。刘碧青在豫章小学上到四年级后去了美国，小时候喜欢画画，奶奶就领她到少年宫去学画画。
记得办好签证的那天，刘溯对女儿说，你的人生可能会发生变化，你要在另一个环境中去成长，不管怎样，希望你保持努力不懈的那种劲头。
后来刘碧青学动画，就去了巴沙狄那艺术中心设计学院。刘碧青学得刻苦，大学学习可以用“疯狂”二字来形容，大学四年，她几乎没有在晚上两点之前睡过觉：光黑色就要学习调200多种不同的黑，一点都不能松懈，正是这样的疯狂奠定了她扎实的基础。
在皮克斯动画工作室工作也是近乎极限，一是她非常认真，每件事情要求做得完美，二是同时担任多部动画片的艺术设计。刘碧青先后担任过《恐龙当家》《海底总动员2》《寻梦环游记》《超能总动员2》等影片的艺术设计和管理工作，是皮克斯年轻且能担大纲的女性骨干。
有人曾经问过她，对获奖有什么想法，她回答：“对我来说，我最想要的是做出一个能够连接父母和儿女、能感动大家的片子。能在中国的电影院里上映我非常兴奋，也很感激，其他的都是额外奖励。”
作为一名艺术设计学院的高才生，能进入皮克斯画动画是她一个很遥远而又很美丽的梦想。刘碧青接到皮克斯公司通知面试的电话时，她居然不相信是真的，以为是恶作剧。面试非常严格，一天时间，十几位动画界如雷贯耳的大咖，轮流面谈，一人一票表决，直接关乎应聘者的命运。
刘碧青进入皮克斯的第一个项目是《恐龙当家》，这部片子历经中途换将，整个团队历经几乎重来的磨难。在公司的电影首映庆典上，很多员工都哭了。进入皮克斯并不是一劳永逸，还需要努力和一些运气。
“我不知道是创作影响了她的情感，还是她的感情影响了创作，但我知道女儿开始长大了，她开始在探究和认识爱的本源，希望可以用作品来传递这份温情。当我们观看《包宝宝》这部短片时，那些温暖的色彩、每一个柔软的细节，已经感受到了这种爱的氛围，无需再用语言去述说。”刘溯说。
