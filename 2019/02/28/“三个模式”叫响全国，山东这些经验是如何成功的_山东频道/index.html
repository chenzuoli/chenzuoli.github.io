“三个模式”叫响全国，山东这些经验是如何成功的_山东频道

发表于 2019-02-28 | 浏览量

原标题：“三个模式”叫响全国，山东这些经验是如何成功的？




乡村振兴再出发，务必坚持干字当头，坚持问题导向，树立更高标准，注重实际实效，创新“三个模式”，助力打造乡村振兴齐鲁样板。
原标题：“三个模式”叫响全国，山东这些经验是如何成功的？
日前，2019年中央一号文件《中共中央国务院关于坚持农业农村优先发展做好“三农”工作的若干意见》正式发布。中央农村工作领导小组办公室主任、农业农村部部长韩长赋解读文件时表示，要“突出实施乡村振兴战略总抓手”，“推动乡村振兴一年一个新进展，不断增强农民群众的幸福感、获得感。”
由中国社会科学院哲学研究所与潍坊市委、市政府等单位联合举办的“诸城模式”“潍坊模式”“寿光模式”与乡村振兴理论研讨会近日在北京举行，国内“三农”领域有关领导、专家、学者共同总结“三个模式”的发展历程和实践经验，探讨“三个模式”推动乡村振兴的策略。
在山东，无论是实践总结还是理论研究，乡村振兴的相关探讨从未停止。首届乡村振兴（山东）高峰论坛曾于去年十月在济南举办，山东省委书记刘家义也曾在多个场合强调要“打造乡村振兴的齐鲁样板”。
诸城模式、潍坊模式、寿光模式，山东这些经验啥内容？对打造乡村振兴齐鲁样板有什么借鉴意义？大众日报·新锐大众记者为您梳理。
2018年，习近平总书记在全国“两会”期间到山东代表团参加审议和视察山东时两次讲到：“改革开放以来，山东创造了不少农村改革发展经验，贸工农一体化、农业产业化经营就出自诸城、潍坊，形成了‘诸城模式’‘潍坊模式’‘寿光模式’。”
“三个模式”是山东从改革开放初期至今不断创新发展的成果，到底都是什么模式呢？
总结来说，“诸城模式”是诸城市在商品经济大合唱、贸工农一体化、农业产业化等不同阶段创新探索成果的总成；潍坊是农业产业化的发源地，“潍坊模式”是潍坊各地农业农村改革创新经验的集成；“寿光模式”则是对寿光在蔬菜产业的生产、销售、技术、会展和标准输出等不同发展阶段改革创新经验的概括。
因为各个地区的具体状况和发展方向不同，“三个模式”的聚焦点和解决方式也有所差异：“诸城模式”解决的主要是农业生产、加工、流通脱节的问题；“潍坊模式”解决的主要是农户分散经营与大市场之间的矛盾；“寿光模式”解决的主要是蔬菜生产产业化的问题。
改革开放奠定了山东农业大省的地位，“三个模式“也伴随着改革开放40年进程一路探索、实践形成。“三个模式”是山东农业发展中从实际经验到理论的总结，不同的历史阶段也有着相应的转变和重点，主要解决的是小农经济与产业化发展之间的矛盾，其实践成果、理论成果、制度成果成为我国“三农”政策创新的重要组成部分，极大地惠及了乡村建设和发展，更为全国其他地区的发展提供了借鉴样本。
“‘三个模式’是各级党委、政府带领广大干部群众用改革的精神、创新的办法破解农业发展瓶颈和矛盾的成功实践。”上个月，潍坊市召开全市农村工作会议，指出要坚定扛起创新提升“三个模式”的政治责任和历史使命，推动乡村振兴走在前列，打造乡村振兴齐鲁样板。
上世纪90年代，潍坊市就在全国率先孕育了农业产业化雏形。“潍坊模式”创新农业生产的组织形式、经营模式和运行机制，从一分实验田发展到全域品牌化。如今，潍坊市农业总产值居全省之首，蔬菜、肉类出口分别占全国10%和12%，创造了一系列在全省全国有影响的经验，并让“潍坊模式”普惠全国。
寿光充分利用自身的蔬菜发展基础，改革开放后通过建设蔬菜批发市场、推广冬暖式蔬菜大棚、举办中国（寿光）国际蔬菜科技博览会、发展现代化大农业等举措成功树立“寿光模式”。据寿光市委书记林红玉介绍，寿光如今新建大棚中物联网职能设施应用率达到80%，农民劳动生产率提升200%以上，自主知识产权蔬菜新品种达到64个，已经开始向全国输出寿光标准和集成解决方案。
诸城市自上世纪八十年代组织开展“商品经济大合唱”，而后又大力推行“贸工农一体化”，逐渐走出了一条产业化经营的路子，逐渐形成了“诸城模式”。据统计，诸城市在2017年完成地区生产总值824.69亿元，是1987年撤县建市之初的78.5倍；全市城镇居民人均可支配收入36998元、农民人均纯收入18670元，分别是1987年的40倍、31倍。
“三大模式”的成功，不仅带动了当地乡村发展，也惠及到周围地区，促进全山东“三农”改革发展。党的十八大以来，山东省以占全国6%的耕地和1%的淡水资源，贡献了8%的粮食产量、9%的肉类产量、12%的水果产量、13%的蔬菜产量、14%的水产品产量和19%的花生产量，农产品出口总额占全国的24%，农业农村现代化建设取得了显著成就，在全国农业发展中占有重要地位。
“三个模式”的成功实践，特别是农业产业化的理论和制度成果，成为指导我国农业和农村经济发展的重要依据，蕴含其中的理念和做法，直到今天仍然具有很强的现实意义和生命力。
潍坊市委书记惠新安说，提升“三个模式”需要结合实际，坚持农业现代化和农村现代化一体设计、一并推进，着力在科技创新、绿色发展、农业开放、融合发展、补齐农村短板、深化农村改革等方面探索实践，不断赋予“三个模式”新内涵，努力创造可复制、可推广的新鲜经验，为打造乡村振兴齐鲁样板。
因此，推动乡村振兴，务必要深刻理解地区实际情况和发展方向，务必要深化对“三个模式”丰富内涵的认识，准确把握创新提升“三个模式”。在深化对“三个模式”内涵和本质认识的基础上，把握好创新提升“三个模式”的重要问题。
一方面，要在乡村振兴大背景下谋划和推动，坚持农村现代化与农业现代化一体设计、一并推进，突出农业供给侧结构性改革；要顺应融合发展大趋势，要着力用好改革开放关键一招，把促进农民增收作为出发点、落脚点，注重转变农村生活方式。
另一方面，要坚持与时俱进，进一步大胆探索，推动创新发展，让“三个模式”展现出新的生机和活力。省委书记刘家义在今年山东省“两会”期间到潍坊团参加审议时表示：要以敢为人先、敢闯敢试的劲头，在新的时代背景下，进一步深化、拓展、创新、提升“三个模式”，为打造乡村振兴齐鲁样板作出贡献，推动高质量发展实现新突破。
乡村振兴再出发，务必坚持干字当头，坚持问题导向，树立更高标准，注重实际实效，创新“三个模式”，助力打造乡村振兴齐鲁样板。
