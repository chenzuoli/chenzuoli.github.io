原标题：陕西：特色小镇星罗棋布 文化助力乡村振兴
新华社西安4月28日电（记者 李勇、梁娟、姚友明）几年前，居住在陕西省泾阳县崇文镇宋村的邢小英，还过着关中平原农民“面朝黄土背朝天”的生活。现在，她在泾干镇双赵村的一家茯茶加工厂工作，已能够熟练操作高自动化的紧压茶生产线，每月收入近3000元人民币。
茯茶是古丝绸之路上的主要贸易商品之一，泾阳是其加工地。邢小英的家乡正发生着深刻改变：从一度大量人口流失的萧条村镇，发展成潺潺流水绕村而过、关中民居错落有致的茶文化特色小镇。“家家都经商，户户有产业”已成为周边村落居民生活的真实写照。
“茶文化”特色小镇建设焕发了茯茶古镇的新活力，离泾阳不远的陕西铜川“瓷器文化”小镇也已成型。从2017年下半年起，铜川市挖掘“耀州窑”的历史文化资源，改造了黄堡工业园区内10多处废弃企业厂房，最终将建成集遗址观光、研发生产、工艺体验和餐饮住宿等功能于一体的“耀瓷小镇”，使历史文化资源、工业遗址和现代陶瓷产业在耀州窑文化基地实现融合发展。
“耀州瓷是中国古代北方青瓷的代表，耀州窑创烧于唐，鼎盛于宋，是中国古代六大陶瓷名窑之一，”耀州窑文化基地管理委员会主任何建平说，“耀瓷小镇将三年成型、五年成城。这里不仅会成为陶瓷手工艺人纷至沓来的宝地，更是向世界展示中国陶瓷文化的窗口。”
文化是特色小镇的“灵魂”，产业是特色小镇的“支撑”。以政府主导下的创新研发为抓手，高附加值的新产品释放了特色小镇原本的文化价值潜能。
西咸新区茯茶镇文化产业集团有限公司董事长魏金娟说：“绿茶、普洱茶、白茶、黑茶、乌龙茶，现在我们都能将其制成茯茶。”在茯茶镇企业的直销点，茯茶突破了它千百年来一成不变的产品形态，茉莉花茯茶、袋泡茯茶、速溶茯茶等产品琳琅满目，受到消费者的喜爱，不少创新茶产品还沿着“一带一路”走出了国门。
铜川市耀州窑陶瓷发展有限公司的闻香青瓷卮杯和青瓷坐狮已成为耀瓷新品中的“网红”。公司董事长蔡云川从2009年起引进中国德化、景德镇等地的技术工人来黄堡创业，攻破了制瓷泥料工厂化生产的技术壁垒，运用3D打印模具技术，“克隆”研发出多款青瓷文创产品，让消费者爱不释手。
“中国传统文化已进入了创造性转化、创新性发展的新时期，千年耀瓷也到了踏上新路再出发的历史节点。”蔡云川说。
西北大学经济管理学院旅游管理系副教授郭亚军说，陕西是中华文明的发祥地之一，数千年的历史文化、红色文化、民俗文化、当代文学艺术、秦岭生态文明等资源驰名中外。“将文化作为小镇的‘灵魂’，将产业与文化进一步融合，推动了陕西省特色小镇的不断更新换代和升级。”
青木川明清古建筑文化与叶广芩小说里故事一样神秘诱人；《白鹿原》影视剧带火了影视城、民俗村和白鹿仓，民俗文化风情浓郁的袁家村、马嵬驿让陕西乡村旅游驰名全国；照金小镇的红色文化与生态文化交融；翱翔航空小镇、西部云谷硬科技小镇等让人看到西安市对战略性新兴产业的强大“西引力”……鲜花小镇、音乐小镇、诗经小镇、温泉小镇等各具特色的小镇也雨后春笋般出现在三秦大地。
郭亚军说，陕西特色小镇星罗棋布，各具特色，让陕西以历史文化为主题的旅游更加丰富多元，也让中外游客有机会深入了解中国传统文化，体验和传播中国传统文化。
中国著名作家贾平凹的旧居、文学作品主人公的原型和家门口的“丑石”，以及茶马古道的丝绸之路历史，是位于秦岭南麓的陕西省丹凤县棣花古镇的“卖点”。
“政府出60%的资金，贫困户贷款解决40%的改造金。”商於古道管委会丹凤县办公室副主任李瑞峰介绍棣花古镇建设情况时说，“改造完成后，政府将改造房屋免费给贫困户经营。140多名贫困人口有的把房子租出去，一年一家人能赚取租金1万余元，自己在小镇里打一份工，又能赚2万多块钱。这样一来，大家脱贫几乎不存在问题了。”
从2013年到2017年，陕西省35个重点示范镇累计完成投资660.76亿元，带动镇区人口增加63.61万人，解决7.26万贫困人口就业问题。31个文化旅游名镇累计完成投资160.93亿元，恢复传统街区28条，吸纳非农就业人数18.96万人。
陕西省副省长陈国强说，陕西省将继续以特色小镇建设作为带动区域性脱贫的重要抓手，致力于实现特色小镇持续健康发展和农村贫困人口脱贫“双重目标”。（完）
