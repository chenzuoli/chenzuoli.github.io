其实，家，才是归途，而房子意味着家，没有房子就像没有家，人在四处漂泊，心也无处安放，更遑论归途，因此，我们总渴望着“房子”。

一个承载着梦想和期望，一个容纳着全家的幸福，一个能给予自己安全感和归属感的地方，那是个什么的地方呢？

，满足你对“家”的所有想象，愿你从此以后，有港湾可避风，有归途可来去，此心安处，是汝乡。
