新房成交量低迷青岛楼市节后“焦虑”状态持续_青岛频道

发表于 2019-02-28 | 浏览量

原标题：新房成交量低迷 青岛楼市节后“焦虑”状态持续
经历过2018年的跌宕，今年年初在诸多不确定因素的作用下，青岛楼市依旧处在一种“焦虑”的状态，进入了己亥年。近日，记者走访获悉，农历春节后，岛城楼市还没有“活跃”起来。一方面大多数开发项目已经处在“整盘”期，很多片区都相对“静默”，没有最新动向。另一方面对于投资置业者来说，在相比以往更客观的对待楼市，何时出手“捡漏”也成为一个至关重要的问题。业内人士表示，面对2019依旧存在很多可能性，但是房价平稳是主旋律，再次“跌宕”是小概率事件。
胶州及西海新区先热
多片区相对“静默”
受节日影响，进入己亥年之后，新房成交量一路低迷，在年后第二周稍有起色，据青岛网上房地产公布的成交数据统计，上周(2月17日-2月23日)青岛新房共成交1826套，环比前一周上涨147.43%，成交面积为209552.92㎡。其中住宅一周成交1510套，占新房总成交量的82.70%，成交面积为183778.44㎡。但从成交量来看已经基本恢复到春节前水平。
从分区成交行情来看,西海岸、胶州、高新区成为热点成交区域。实际上记者了解到，对于开发企业来说，这些成交量正式片区内开发企业“积极互动”的结果，记者走访获悉，新春之后，胶州西海岸的开发企业相对其它片区更为积极。尤其是胶州，多楼盘推出最新消息。西海岸红星天铂、胶州荣盛锦绣外滩、花样年碧云湾等项目都已经开始行动。
相比之下，即墨新区及传统意义上的主城区内的项目表现并不“积极”。很多项目目前依旧没有进入“备战”状态。并没有出现最新的动态。市北一楼盘的营销总监告诉记者：项目目前还没有最新的动态，近期的活动最早会安排在一季度末。
置业者客观“应对”
“看菜下单”的情况下何时出手依旧焦虑
记者在走访中看到，对于今年楼市的走向，很多购房者参照去年的情况，已经客观很多。但是究竟什么时候“出手”依旧是个问题。“在2018年年末的时候，很多人都说2019年的楼市不乐观，至少在上半年会一直持续低迷状态，而且从政策层面来看，楼市很明显不会出现大的动荡了”，从事房地产投资多年的刘先生对今年的房产投资依旧不乐观。
刘先生告诉记者，往年每到年底朋友聚会的时候谈起房产投资都会“眉飞色舞”，但是今年大家都“低调”很多。刘先生的一位朋友在去年年中投资的一套房产，年底时收益明显低于预期。“房产投资不像前两年了，随便找个新规划区域投投就可以了。现在的投资者必须擦亮眼睛，一不小心就会掉进‘坑’里了”，刘先生如今房产投资不再只是“有钱人的游戏”了。
尽管如此，刘先生依旧在寻找新的投资契机。“对于青岛来说，作为时尚、开放的国际化大都市，机场、地铁、高铁、海底隧道交通配套处在大建设阶段，很多高校纷纷落子、新兴片区的配套也逐渐完善，很多潜力区域有待开发，投资环境还是有的，我还是很有信心的。只是把握时机非常重要。” 记者了解到，对于投资者来说，今年依旧会以基建及规划为导向，郊区一些总价低的潜力区域也成为关注的焦点区域。
对于刚需来说，2019依旧是买房年。房地产专家孟祥远在接受采访时曾表示，今年三四月，楼市有望再入春，他认为在2018年下半年，“央行”的放水行为已经是明显，多个城市回调首套房贷利率，甚至部分城市还放松了二套房的房贷利率。利率上调能有什么影响？从买房者个人而言，还款压力大大减轻，从购房者整体而言，刺激了购房者的购买欲望，从而加快楼市去库存。再加上下半年激烈的抢人大战，在限购的政策之下，落户就相当于有了购房资格。两重利好的刺激，越来越多的观望者加入买房大军的队伍。楼市自然也就回温进入春天了。
开发企业联手拿地“减压”
家里有地心里不慌
与往年类似，在推盘节奏不快的情况下，土地交易的节奏加快。岛城近期多个地块已经出让成功。进入2月份以来，绿地以9.46亿元竞得城阳326亩地，青岛实昊星置业股权转让，涉及西海岸住宅用地10万方，青特置业首进即墨，即墨3宗地卖出5个亿；西海岸35万方地块上市，起拍总价8.6亿。
此外，融创85亿美元新项目落地，西海岸88万平方米地块卖出27亿；接下来世茂胶州首个项目落地，胶州48万平方米住宅卖出8.3亿元。世茂、天泰、及胶州新城等纷纷落子。再有蓝光与融创分食西海岸两宗地；青岛地铁蓄势而发以2.07亿元拿下市南1宗地。 2月26日，2月26日，青岛远景置业有限公司,青岛地铁资源开发有限公司和青岛丽达商业发展有限公司以19.3亿的总价竞得浮山后铁艺街北侧地块，楼板价7100元/㎡。
从目前的土地交易情况来看，融创、世茂等去年的“明星”房企活跃度不减，一些发展势头的较好的本地房企也成为拿地的高手，青特、青岛地铁都蓄势待发。从房企拿地的情况来看，2019年热点区域将继续“外扩”。
值得注意的是，房企联合拿地的情况显著。有专家表示，从2019年开始国内土地市场将要发生一些本质的改变：对于还想简单拍地盖房快消的房企，拿地将变得非常困难，大多规模小、品牌小、专业弱的房企前途就堪忧；但对于有钱有资源内生力十足的房企，“家里有地心不慌”，新的一年还将上演一场土地盛宴。
专家预测，在市场不好、房企分化越来越严峻的2019年，大鱼吃小鱼、现金多吃现金少，已经是必然趋势，一场并购大浪潮呼之欲出。中小房企即使熬过2018，或许再也很难熬过更艰难的2019，这注定房企2019年逐渐成为房企并购的爆发年，房企联合拿地的现象将不再是个别现象。对于整个楼市来说，面对2019依旧存在很多可能性，但是房价平稳是主旋律，再次“跌宕”是小概率事件。（记者 葛均艳）
