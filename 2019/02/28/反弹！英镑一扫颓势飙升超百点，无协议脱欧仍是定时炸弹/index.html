反弹！英镑一扫颓势飙升超百点，无协议脱欧仍是定时炸弹

发表于 2019-02-28 | 浏览量

近两周来，英镑随着硬脱欧风险下降、脱欧期限有望延迟以及英国央行可能实施“渐进式加息”诸多利好因素的提振而持续上扬，一举成为目前外汇市场上最受瞩目的货币，一扫此前颓势。



薛皎
近两周来，英镑随着硬脱欧风险下降、脱欧期限有望延迟以及英国央行可能实施“渐进式加息”诸多利好因素的提振而持续上扬，一举成为目前外汇市场上最受瞩目的货币，一扫此前颓势。
不过，眼下交易员们虽忙得不亦乐乎，但脱欧的这颗“定时炸弹”仍然影响着市场各方的决断，随时可能引爆。
“在当前的乐观情绪中，英镑有望延续涨势，但问题是能持续多久呢？”FXTM富拓研究分析师卢克曼·奥图纳对第一财经记者称，英镑已对脱欧相关消息变得极度敏感，英镑走势可能随着3月29日的最后期限临近而更加震荡。
2月28日，英镑对美元延续了此前10个交易日的整体上涨趋势，盘中再次突破1.33关键点位，盘中最高报1.3326，涨幅达到1.4%。目前，英镑维持在近8个月的高位上，英镑两个月隐含波动率也接近2018年11月低点。
此前，连续多日的强劲反弹已经令英镑备受交易员关注，甚至预计其有望成为今年表现最好的货币。年初以来，英镑对美元、欧元已经分别上涨了约4%。
“现在终于到了交易价格而不是脱欧新闻的时候了。”来自英国的资产配置策略师克里斯·杰弗里表示。他掌管着9850亿英镑资产。他认为，虽然最新的脱欧进展仍需要关注，但目前英镑的价格已经成为最主要的判断依据。
近期英镑反弹的主要动力，来自于无协议脱欧风险的下降以及脱欧日推迟的可能。
2月24日，英国首相特蕾莎·梅表示，将原定于2月27日脱欧协议第二次表决投票推迟到3月12日举行。
随后，梅又在2月26日承诺，如果脱欧协议在3月12日英国议会下院投票中未获通过，且“无协议脱欧”被排除，那么将由议会下院通过投票决定是否推迟“脱欧”。
受到这一消息推动，英镑对美元在1个小时内快速上涨约100点，日内涨超130点，英镑对美元上升至1.33关口上方，为2018年9月以来首次。
同一天，英国央行行长马克·卡尼也提出，如果经济表现符合预期，英国需要渐进式加息。此外，卡尼称英国央行加强了对于英国紧急退欧的防御措施，在3月退出日期前后提供额外的流动性准备金。受卡尼讲话提振，英镑对欧元创下21个月新高。
值得注意的是，英镑的上涨还离不开近期美元弱势表现。
2月27日，美联储主席鲍威尔在国会证词中重申“耐心”的政策论调，令美元受到打压。
尽管鲍威尔对美国经济状况的评估较为乐观，但他承认面临国内外多种不利因素。美国此前公布的一系列经济数令人失望已经引发市场对经济增长前景的担忧，地缘政治风险和全球增长停滞担忧更是加剧了这种不确定性。保持“耐心”和“灵活性”的美联储可能在较长时间内保持利率不变，美元仍面临下行风险。
奥图纳认为，美元下跌是对美联储官员言论和经济数据敏感性增强的早期信号。“总而言之，美元多头的动力正在逐渐耗尽，这继续反映在美元指数走势中。”
尽管短期内硬脱欧的风险下降，脱欧日期推迟的可能性上升，但脱欧这颗定时炸弹以及随之带来的不确定性，对经济以及市场信心的影响，远未解除。
德意志银行在最新报告中，将英国无协议脱欧的可能性由15%下调至10%，但上调举行大选的可能性至20%。
英国议会党派的纷争，梅首相退欧政策一变再变，令社会各界对于英国顺利脱欧的信心严重受挫。
英国智汇外汇交易公司首席市场分析师纳伊姆·阿斯兰姆（Naeem Aslam）对第一财经记者说道：“梅的脱欧政策像旋转门一样不停改变方向。她把国家的命运推到了悬崖边上。英国议会这个政治马戏团，眼下成了一个真正的笑话。”
阿斯兰姆认为，尽管无协议脱欧的可能性下降给市场注入了很多乐观情绪，但这个选项，其实仍然在谈判桌上。
对于3月29日退欧日期推迟的可能性，英国商业给出的反应也是喜忧参半。英国工业联合会（CBI）总干事卡罗琳·费尔拜恩认为，尽管推迟比在几周内不达成协议就崩溃要好，但推迟脱欧面临更多的不确定性和成本增加的风险，企业必须相应的调整应急计划。“目前妥协是唯一的办法。”费尔拜恩说道。
瑞士工业设备制造商abb有限公司的英国总经理伊恩·芬内尔则直言，即使需要面对无协议脱欧，也宁愿按原计划在3月底脱欧，而不是将不确定性再延期两个月甚至更长。
眼下，无论脱欧结果如何，对于英国经济造成的结构性影响，恐怕短期内已难以改变。
瑞银财富管理最新调查发现，英国投资者已成为欧洲最悲观的市场参与者。英国脱欧对信心产生了破坏性影响，超过一半的受访者认为国内政治是他们最关心的问题。唯一一个同样令人担忧的市场是意大利。瑞银财富英国国内业务主管尼克•塔克表示：“英国投资者在几乎所有方面都大幅降低了对2019年的预期。”
2月7日，英国央行下调了英国2019年经济增速预期，该经济增速有可能成为10年来最低水平。英国央行并预计今年上半年英国有四分之一的可能性进入衰退。

首相办公室在动议被否决之后，低调表示政府会持续与布鲁塞尔协商，想办法修正现有的脱欧条款。
英国与瑞士11日签署贸易协议，以求在英国“脱欧”后继续维持两国间现有的贸易优惠条件。
周末特朗普发表的“重要讲话”让投资者失望后，周一，市场正静待英国首相特雷莎·梅公布英国脱欧的“B计划”，欧洲各国股市走势亦随之出现分歧。
市场普遍预计，梅政府会在15日的议会投票中落败，这种可能性已大部分体现在近期英镑的定价中，因此很难看到大幅波动。
美国政府关门时间接近历史最长的消息，引发投资者担忧。有报道称，美国白宫管理和预算办公室（OMB）就联邦政府停摆可能会持续至2月末这一问题正在制定计划书。
