山东省委第四轮巡视公布问题清单发现这些问题_山东频道

发表于 2019-02-28 | 浏览量

原标题：省委第四轮巡视公布“问题清单”，这些单位发现这些问题

根据省委巡视工作领导小组安排，近日，省委巡视组向被巡视单位反馈巡视情况。以下这些被巡视单位，发现这些问题。
原标题：省委第四轮巡视公布“问题清单”，这些单位发现这些问题
根据省委巡视工作领导小组安排，近日，省委巡视组向被巡视单位反馈巡视情况。以下这些被巡视单位，发现这些问题。
济宁市
近日，省委第一巡视组向济宁市委反馈巡视情况。据《省委第一巡视组向济宁市委反馈巡视情况》显示，近年来，济宁市委在学习贯彻习近平新时代中国特色社会主义思想和党的十九大精神，坚决维护习近平总书记核心地位、维护党中央权威和集中统一领导，履行全面从严治党主体责任，贯彻新发展理念等方面取得积极成效，各项工作取得一定进展。巡视中，巡视组发现和干部群众反映了一些问题，主要是：贯彻落实习近平新时代中国特色社会主义思想和中央重大决策部署成效不够明显，落实绿色发展理念不够坚决，推进新旧动能转换重大工程措施不力，意识形态工作存在薄弱环节；党建工作基础薄弱，选人用人机制不够健全，净化政治生态任务艰巨；坚持以人民为中心的发展理念树得不牢，招商引资存在“声势大、效果差”的现象，扫黑除恶、退役军人工作推进不力，侵害群众利益的腐败问题依然屡禁不止；作风建设任重道远，不担当不作为乱作为现象时有发生，文山会海、考核过多过滥问题依旧存在，反腐败斗争形势依然严峻复杂；巡视整改存在重部署轻落实，巡视反馈问题整改不到位的问题。同时，巡视组还收到反映有关领导干部的问题线索，已按规定转相关纪检监察机关或组织部门处理。
济宁市兖州区
2019年2月21日，省委第一巡视组向济宁市委通报巡视兖州区情况；2月22日，向兖州区委反馈巡视情况。据《省委第一巡视组向济宁市兖州区委反馈巡视情况》显示，近年来，济宁市兖州区委在学习贯彻习近平新时代中国特色社会主义思想和党的十九大精神，坚决维护习近平总书记核心地位、维护党中央权威和集中统一领导，履行全面从严治党主体责任，贯彻新发展理念等方面取得积极成效，各项工作取得一定进展。巡视中，巡视组发现和干部群众反映了一些问题，主要是区委领导核心作用发挥不充分，贯彻中央重大决策部署力度有差距。推动新旧动能转换工作措施不实，脱贫攻坚工作落实不到位，防范化解金融风险不力，意识形态工作存在薄弱环节；党建工作基础薄弱，干部队伍建设有短板。党的组织生活制度落实不严格，民主集中制执行不规范，基层党组织作用发挥不充分，领导班子和干部队伍建设存有短板；全面从严治党“两个责任”落实不力，“四风”问题禁而未绝。压力传导层层递减，违纪违法案件多发，“文山会海”问题更加突出。违反中央八项规定精神问题时有发生；持续抓整改意识不强，巡视“后半篇文章”质量不高。同时，巡视组还收到反映有关领导干部的问题线索，已按规定转相关纪检监察机关或组织部门等处理。
汶上县
2019年2月21日，省委第一巡视组向济宁市委通报巡视汶上县情况；2月22日，向汶上县委反馈巡视情况。据《省委第一巡视组向汶上县委反馈巡视情况》显示，近年来，汶上县委在学习贯彻习近平新时代中国特色社会主义思想和党的十九大精神，坚决维护习近平总书记核心地位、维护党中央权威和集中统一领导，履行全面从严治党主体责任，贯彻新发展理念等方面取得积极成效，各项工作取得一定进展。巡视中，巡视组发现和干部群众反映了一些问题，主要是县委对全县经济社会发展用力不够，落实中央重大决策部署有差距。全县干事创业氛围不浓,推动转型发展效果不明显,乡村振兴战略落实慢,意识形态工作责任制落实不到位;党建工作不扎实，干部队伍建设存在薄弱环节。党内组织生活不规范、不严肃,干部队伍建设存在薄弱环节;作风建设不严不实，整治“四风”仍有差距。不担当不作为现象时有发生,违反中央八项规定精神和“四风”问题一个时期易发多发;全面从严治党“两个责任”落实不到位，腐败问题易发多发。关键岗位和重点领域腐败问题易发多发,村级“三资”管理不规范，侵害群众利益的问题屡禁不止;巡视整改不坚决不彻底，与做好巡视“后半篇文章”要求有差距。同时，巡视组还收到反映有关领导干部的问题线索，已按规定转相关纪检监察机关或组织部门等处理。
泰安市

宁阳县

临沂市
