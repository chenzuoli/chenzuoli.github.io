游戏业冰火两重天世纪华通海外业务亮眼

发表于 2019-02-28 | 浏览量

经历了去年行业版号暂停、商誉减值等风波，游戏行业有些烦，随着近期财报预告纷纷公布，也让外界窥见了游戏行业过去一年的成绩单以及发展趋势。


何天骄
经历了去年行业版号暂停、商誉减值等风波，游戏行业有些烦，随着近期财报预告纷纷公布，也让外界窥见了游戏行业过去一年的成绩单以及发展趋势。
2月28日， A股最大游戏公司世纪华通（002602.SZ）发布了业绩快报，2018年实现营业收入80.74亿元，同比增长131.29%；归属于上市公司股东的净利润10.06亿元，同比增长28.48%。
世纪华通方面表示，业绩增长主要是由于点点互动并表带动。据悉，点点互动主要布局海外游戏业务，海外业务的良好表现让世纪华通利润增速领跑A股游戏企业。此外，随着世纪华通收购盛大游戏成功过会，增长空间也将不断扩大。目前，随着国内游戏行业低迷，让很多游戏公司将目光转向海外，而一些先期布局海外的游戏公司在这个时候尝到了甜头。
游戏行业研究专家洪涛向记者表示，目前看以海外市场为主要业务的上市公司，比如世纪华通、宝通科技、IGG等公司，整体表现好于其他厂商，甚至个别公司业绩增速超预期。世纪华通CEO王佶也认为，游戏出海是目前游戏公司需要重点布局的。在他看来，海外市场更具有更广阔的潜力。
但并非所有游戏企业的业绩都向好。今年1月，一大批游戏公司发布了不甚理想的业绩预告。目前游戏股中，仅1月份就有天神娱乐、游久游戏等16家A股游戏上市公司发布业绩预亏，超过10家公司提及了商誉减值的影响。其中，最典型的案例是天神娱乐，预亏金额高达78亿元，其中计提商誉减值49亿元。此外，天神娱乐设立参与并购基金及股权投资资产预计分别减值15亿元及7.5亿元。抛开资产减值影响，天神娱乐2018年归母净利润预计为3.77亿元。
除了计提商誉减值带来的业绩亏损，主营业务收入大幅下滑也是这次不少游戏公司大幅预亏的原因。计提商誉减值背后，是2018年游戏行业大环境变化，游戏版号经历了9个多月的冻结，导致很多游戏公司主营业务亏损，从而成了不少上市公司隐藏的业绩雷。
洪涛向第一财经表示：“从最近上市公司的财报来看，不少受到了2018年机构改革导致的游戏审批暂停的影响，新游戏未能按时发布导致了营收增幅的降低，而步入2019年后，随着审批的重新恢复，市场开始关注已获得版号的精品游戏厂商业绩表现，近期来看，三七、完美世界关注度较高，也是体现了这点，在产品供应尚未恢复的市场环境下，能商业化的精品游戏对市场影响非常大。”财报显示，三七互娱在2018年全年实现营业收入76.33亿元，同比增长23.33%。实现归属于上市公司股东的净利润10.48亿元。
不过，在王佶看来，游戏行业寒冬其实只是表面现象。在他看来，游戏正在经历去粗取精的过程，前景依旧一片大好。一方面，很多人认为政策开始收紧，其实是政府机构的调整，带来游戏审批的暂时放缓。第二个是随着游戏精品化趋势越来越明显，游戏行业也在加速整合，去粗取精，一批实力不济的公司就将离场。在王佶看来，未来游戏产业精品化是大势所趋，水准不高的产品很难在市场存活，甚至上线都很难，此外，服务也越来越重要，用户反馈要能够及时地回应和处置。
但不可否认的是，游戏行业的冬天仍在。尤其是去年版号暂停，一大批游戏公司陷入无米之炊的困境，不少从业者被迫转型。近期，又有中国网络游戏审批申请暂停受理的消息在网络流传。不过，记者昨日从广东、上海等地上市游戏公司了解到，目前游戏审批仍在继续，尚未收到审批申请暂停受理的消息。上海市新闻出版局（上海市版权局）方面目前还在正常接受申报材料，只是暂时不寄北京。因总局可能对申报材料的格式有调整，等调整好了会通知恢复。
另一方面，高商誉问题也将减轻。“目前游戏板块上市公司与其他行业企业步调较为一致，在2018年内处置了商誉风险，这也导致部分公司的净利润出现了下滑，但有助于2019年更好体现业务内生增长的动力。”洪涛表示。

浙江世纪华通集团股份有限公司发行股份及支付现金购买资产并募集配套资金暨关联交易事项获得有条件通过。

“你觉得今年中小游戏公司生存艰难，可我想说，明年还会更艰难，只有能够做出爆款的巨头和在细分领域深耕做出好的产品的企业能够存活，市场会给出公正的结果。泡沫挤掉总是好事，游戏终归是个创意行业。“
