南海网、南海网客户端澄迈2月26日消息(南海网记者高鹏)赌场设在澄迈县白莲、大丰地区的深山老林，出入仅有一、两条路，还有人放哨，每天只有下午开赌，赌客车接车送，不熟的人不让进……尽管反侦查意识高，但仍逃不过警方打击。2月21日晚，这个在深山老林开设赌场的团伙13人在聚餐时被澄迈县公安局老城分局“一锅端”。
深山老林中设赌陌生人员或车辆进入立即报信
2018年12月，澄迈警方多次接到群众举报，有人在澄迈县白莲、大丰地区的山上树林内聚众赌博。两地派出所尽管多次出警，但一到群众举报地点时，可以看到骰子、赌“大小”所用的桌板等赌博工具，却无法抓到涉赌人员。
“这赌博团伙的赌博地点都在深山老林里，出入仅有一、两条土路，我们民警不管是开警车还是私家车去，每次都扑空。”白莲派出所副所长赵开振说。
澄迈县公安局党委对此高度重视，县委常委、县公安局长潘秋光指示，务必严厉打击、彻底铲除。澄迈县公安局老城分局抽调精干警力成立专案组展开侦查。
专案组民警在开展外围侦查时发现，这个赌博团伙非常警惕，有很强的反侦查意识，该团伙选择的地点基本上都是山上的树林里，进出不易，在进出赌博场所的路上都有人放哨，一旦有陌生人员或车辆进入时，放哨的人员立即通过对讲机或微信语音的方式通知，赌博团伙随即分散离开。
参赌人员全省各地专车接送一天设赌2-3小时
“我们为了摸清该团伙的相关情况，民警乔装成赌客，尝试接近该团伙，但该团伙却不予理睬。”澄迈县公安局老城分局刑警中队长冯乃泽说。
民警没有放弃，通过多方努力，终于以赌客的身份接近了该团伙。民警发现，该团伙组织架构明确，分工明确，“摇骰子”、管理赌资、现场付钱、放哨、开车接送赌客等各环节均有不同的人员专门负责。为躲避打击，该团伙不定时变换赌博窝点流窜作案，每天下午2点多-3点开赌且只“营业”2-3个小时。
“这个团伙每次赌博的地点都不一样，赌客只能联系上司机，开赌前连司机都不知道赌博的具体地点。前来赌博的人员分布在澄迈、海口、屯昌、文昌、昌江等地，团伙的司机负责专门接送赌客，新的赌客要想参与赌博需要老的赌客介绍，并由司机转告庄家且得到同意后才能加入。”冯乃泽说。
一赌客两小时输3万多元赌博结束后团伙聚餐
民警在侦查中发现，尽管赌博地点简陋，参赌人员却不少，每天基本都有10多人前来赌博，曾经有1名赌客在两小时内就输了3万多元。
“从摇骰子到结束，一分钟就可以完成一把赌博，设赌人员规定一把最高押注2000元。”冯乃泽说，这个团伙一天几小时的赌博最多赌资达到10万元，平均下来，每天的赌资约5万余元。如果赌客带来的现金不够，管理赌资的专人还会为其刷信用卡或微信、支付宝套现，让他们继续赌博。
经过侦查，民警发现该团伙每当赌博结束后都会一同聚餐，民警和其他赌客曾经想跟该团伙“蹭饭”，但始终没能成功。
聚餐时被“一锅端” 庄家自称赌博欠债
经过前期的大量侦查，收网时机终于成熟，2月21日晚，该团伙组织“全体工作人员”聚餐，当天21时，澄迈警方调集警力守住该团伙聚餐饭店的门口、楼梯。
“当我们冲进他们聚餐的包厢并表明身份时，这些涉赌人员当时都懵了，可以说是集体傻眼了。”冯乃泽说，行动共抓获犯罪嫌疑人13名，查扣涉案车辆2辆，缴获赌具骰子9枚、对讲机6部、手机一批、赌资68380元。
根据前期摸排的情况，民警在对负责“摇骰子”的28岁犯罪嫌疑人姜某晓审讯时，他起初称自己只是给庄家打工的，还胡乱编了个不存在庄家的“外号”，企图干扰警方的办案，但在大量证据前，姜某晓最终承认了自己就是赌博团伙的庄家，占95%的股份，另一名负责管理赌资的30岁犯罪嫌疑人周某安则占另外的5%股份。
冯乃泽说，姜某晓原本是一名计算机老师，而周某安是在老城地区从事海鲜生意，两人在之前的赌博时相互认识。姜某晓虽然自己结婚也有了孩子，但由于好赌，欠下了大量债务，为了还债，他就想自己设赌，还找来了周某安来帮忙看场。周某安见到姜某晓通过设赌赚钱，他也想参与，经过商量后他也参与设赌。
为了逃避警方打击，姜某晓、周某安还专门研究了警方的侦查方式，加上姜某晓是澄迈白莲人，他熟悉这一带的山林，每次由他选点，而且还找来自己的亲戚等人参与放哨、接送赌客。
目前，姜某晓、周某安等13名涉赌人员已全部被刑事拘留，案件正在进一步办理中，赌资也在进一步追缴中。
