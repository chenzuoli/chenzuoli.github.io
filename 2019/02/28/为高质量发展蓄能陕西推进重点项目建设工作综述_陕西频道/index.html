为高质量发展蓄能陕西推进重点项目建设工作综述_陕西频道

原标题：发力“重特大”项目
为高质量发展蓄能
——我省推进重点项目建设工作综述
本报记者 程靖峰
2018年，我省600个省级重点项目完成投资5617亿元，超额完成年度目标任务，为全省经济持续健康发展提供了有力支撑。
2013年至2017年，省级重点项目由380个增加至600个，完成投资增长65%，占同期全社会固定资产投资的比重保持在五分之一左右。210个关系到全省人民福祉的重点项目竣工。
近年来，我省不断扩大有效投资，全力抓好科技创新、先进制造业、基础设施、生态环境、民生和公共服务等领域重大项目，发力“重特大”项目，引领创新驱动发展、产业结构调整和“三个经济”发展，助推陕西追赶超越。
2017年8月30日，省政府、西安市分别与三星电子签署合作框架协议。根据协议，三星电子在高端存储芯片一期项目总投资100亿美元的基础上，再次投资70亿美元建设二期项目。7个月后，二期项目正式开工建设，创造了重点项目推进的“陕西速度”和“西安效率”。
三星高端存储芯片项目的高效推进，源于省市有力保障、部门联动服务。省级领导多次深入项目建设单位，督促检查工程进展。省级各部门切实发挥各自职能作用，形成了齐抓项目、共促投资的良好局面。
在重点项目推进过程中，我省逐步完善了省级领导联系重点项目制度。2013年至2017年，省级领导联系重点项目超过100个，完成投资超过1000亿元，一批建设上存在问题的项目在省级领导的协调下取得积极进展。
为形成合力，推进重点项目建设，我省充分发挥重大项目联席会议机制作用，并由省发展改革委牵头，财政、自然资源、生态环境等部门通力合作，积极协调解决重大项目落地中遇到的问题。
在要素充分保障和部门高效联动支持下，以三星高端存储芯片、CEC·咸阳8.6代液晶面板生产线项目为代表的“高精尖”项目在陕西开花结果。2013年至2017年，我省共实施科技创新类产业项目276个，投资占全省固定资产投资比重由1.08%提升至2.28%。中科院西安科技园、西工大翱翔小镇等新型创新平台加快建设，推动陕西从科技大省向科技强省跨越转变，创新型省份建设初具规模。
华为西安全球交换技术中心、美光科技半导体封装测试工程、强生全球供应链生产基地（一期）、奕斯伟硅产业基地等重点项目相继投入运行或开工建设，我省软件及信息技术、高端半导体芯片、医药制造等战略性新兴产业取得突破发展。
科技创新项目建设不仅带动即期投资快速增长，而且建成后还为经济社会发展提供持续不断的动力。例如，三星高端存储芯片项目吸引并带动了100多家韩国企业和相关半导体企业落户，使西安形成了较为完整的半导体产业链。2018年前三季度，三星进出口额同比增长54.1%，成为拉动西安进出口增长的重要因素。
“我们将通过开展冬季施工攻坚战，加快项目建设进度。”2018年10月23日，延长石油集团炼化公司项目指挥部副指挥吉李彬告诉记者，延长油田伴生气资源循环利用项目冬季施工方案已完成，他们还提前采购了帆布、棉被和电热毯等物资，保证冬季零下20摄氏度也能正常施工。
延长油田伴生气资源循环利用项目是延长石油集团首个聚焦天然气分级分质利用的精细化工项目。该项目的建成，将使企业自产的伴生气、甲醇变为具有更高附加值的化工产品，这对企业优化产业结构意义重大。
优化产业结构不仅是企业转型升级的迫切需求，也是区域经济发展的必由之路。2018年下半年，江苏悦达集团携手江苏苏美达集团、铜川产业基金在董家河投资建设年产600万只汽车轮毂项目。一批铝合金产业链延伸项目相继在铜川市快速聚集，使得当地长时间处于低谷的铝制造业再现活力。
我省以优质项目供给促进产业结构优化。在重点项目的前期论证和后期评价环节，省发展改革委对项目资源代价、环境容量、社会就业、财政税收、投资强度、综合带动（产出能力与科技创新）等进行量化评价、综合打分，作为衡量项目质量的基本依据。按照“重大型、引领型、基地型”项目优先的原则，论证遴选、研究确定产业项目。
优质项目供给加快了产业结构优化。在重点项目布局环节，我省按照关中协同创新、陕北高端能源化工、陕南绿色循环发展的定位，引导企业根据3个区域资源禀赋确定重大投资项目。
2013年至2017年，我省共实施重点产业项目1089个，完成投资5737.3亿元。其中，非能工业项目564个，投资占全省固定资产投资比重由1.98%提升至2.86%；先进装备制造类工业项目132个，投资占全省固定资产投资比重由0.28%提升至0.39%。
延安煤油气资源综合利用等重点项目加快建设或建成投运，使我省煤制烯烃年产能超过300万吨，居全国第一方阵；吉利汽车产业基地和发动机等汽车项目建成投产，带动全省整车产销大幅增长，新能源汽车规模稳居全国前列；随着延安圣地河谷文化旅游产业园区等重大文化旅游重点项目的建成投运或加快建设，文旅产业带动全省服务业加快发展。
2018年4月26日，菜鸟西北物流中心投运；2018年12月18日，圆通速递西北航空转运中心投运；2018年12月25日，顺丰速运西北航空枢纽投运……2018年，国内龙头物流企业在陕布局的物流项目纷纷落地，有力地推动了我省“三个经济”发展。
此外，苏宁西北地区电子商务运营中心、京东西北电子商务基地、国美西北电子商务运营中心等物流项目建成或加快建设。
优良的营商环境为国内物流龙头企业重点项目落地建设提供了坚实保障。仅2018年上半年，我省累计减轻企业负担649.5亿元，新设企业办理营业执照、办理涉税许可事项办结时间大幅压缩。省发展改革委重点抓好省公共信用信息平台和信用中国（陕西）网站建设，推动建立信用协同监管和联合奖惩机制。
在为重点项目创造优良环境的同时，省级相关部门千方百计为项目建设保驾护航。省发展改革委加强与国家部委的衔接，争取资金和政策支持省内重点项目建设；省自然资源厅把保障重点项目用地需求作为工作的重中之重，切实做好土地供应；省生态环境厅主动服务，从源头参与重大项目环评工作，为项目早日落地建设提供支持。
2019年1月11日，国家发展改革委正式批复西安咸阳国际机场三期扩建工程项目建议书。这标志着西北民航有史以来最大的基础设施项目前期工作取得重大进展。在项目推进过程中，省发展改革委坚持挂图作战、压茬推进，攻克了诸多难题。特别是在机场建设审批层级高、程序流程复杂、协调难度大的情况下，项目立项审批仅用时70多天，创下同等规模机场项目立项审批新纪录。
随着一批事关“三个经济”发展的重点项目落地，陕西区位、科教、资源、文化等优势更加凸显。2013年至2017年，我省共实施重点基础设施项目289个，投资占全省固定资产投资比重由2.66%提升至3.13%。新建成铁路、高速公路里程均超过1000公里，高速公路通车里程达到5279公里，高速铁路通车里程达到854公里。截至2018年12月20日，西安咸阳国际机场年货邮吞吐量突破30万吨，2018年累计增速排名保持全国十大机场首位。
