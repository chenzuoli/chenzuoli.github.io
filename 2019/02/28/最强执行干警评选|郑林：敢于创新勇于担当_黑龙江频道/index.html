最强执行干警评选|郑林：敢于创新勇于担当_黑龙江频道

“24.2万元，成交！”2018年6月24日，黑龙江省宁安市人民法院的拍卖现场，随着一件被执行人的特殊财产——手机“靓号”——竞拍成功，该院执行警务局局长郑林悬着的心落了下来。在郑林的促动下，这次“特殊”的竞拍开了牡丹江两级法院的先河，也让郑林的“执行艺术”有了最直观的展现。
敢创新，让执行方式多起来
执行，对于申请执行人而言，是实现公平正义的“最后一公里”。对于郑林来说，要实现这样的目标，还需要不断的创新。“执行工作要跟上时代的步伐”，这是郑林的执行理念，也是这样的理念促成了24.2万元的竞拍。
这起竞拍的起因是一桩建设工程合同纠纷案件。案件审结后，留给郑林和执行干警的是被执行人正在服刑，其名下唯一房产已被多家法院查封的难题。困境前，郑林和他的团队一度停滞难行。可申请执行人的利益等不起，郑林和干警几次翻阅案卷，了解被执行人的财产情况，一个不经意的发现给了郑林“灵感”。
“我们发现被执行人手机号是吉祥号，这可能是执结这个案件的一个突破点。”于是，郑林召开执行长联席会议，将手机号码作为被执行人的虚拟财产进行评估、拍卖。竞拍被执行人的手机号码在牡丹江“史无前例”，“做好这次拍卖对我们来说意义重大，这关乎着申请执行人的利益能不能实现。”郑林说。
郑林团队对手机号码进行了评估，最终将评估价确定为6万元。“这么高的价格会不会有人竞拍？”这是郑林的疑虑。在竞拍现场，44轮竞价，不断为郑林的信心加码。最终，这个特殊的财产最终以24.2元拍出。
“只有创新执行方式，用好每一种措施，才能让执行‘不难’。”郑林感慨道。
敢担当，让执行工作“暖”起来
在郑林看来，执行需要创新，也需要温度。
2016年，由于无法按期拿到工程款，鞍山一锅炉厂将宁安市某农业开发公司告上了法庭并最终胜诉。可进入执行程序后，郑林遭遇了重重困难。
“一边是外地来的投资商，如果不执行到位，势必会冷了投资商的心，对营商环境也是‘重创’；另一边是本土企业，如果强制查封、拍卖资产，本地企业未来可能‘回天乏术’。”摆在郑林面前的是“手心手背都是肉”的两难，可郑林还是想找到“双赢”的办法。
不断地翻阅案卷和走访后，他发现被执行企业不能按时交付工程款的症结在于经营不善。于是，经执行长联席会议讨论后，郑林决定为被执行企业联系合作伙伴，用“造血”的方式提高企业的履行能力。很快，这样的执行方式得到了双方的认同。
如今，宁安这家企业不仅还上了剩余的工程款，还成为全市民营经济发展的重要力量。“如果不是郑林敢作出这样的执行决定，我们企业就不会有今天。”被执行企业负责人说。
肯担当又能因势利导，郑林的执行艺术，不仅维护了当事人胜诉的合法权益，更优化了营商环境。
敢碰硬，让执行动作快起来
“恶坝”是宁安人对违法堤坝的“俗称”。每当雨季来临，因“恶坝”堵塞河道，两岸农作物便会被洪水淹没，农民一年的辛劳也随之东流。
2017年9月，郑林和执行干警开展了“除恶坝”行动。可长期存在的“恶坝”给郑林带来了极大的挑战。吴某便是其中一个违法堤坝的建设者。败诉后，吴某始终不履行自己的法定义务，甚至和执行干警耍起了小聪明。一次，在执行干警拆除违建堤坝时，吴某装作心脏病发作，躺在了警车上。
时间越拖越久，执行工作停滞不前。吴某可以用耍赖皮的方式拖下去，可遭受损失的农民却等不了。郑林再次施展执行艺术，他联系当地医院，对吴某进行全面检查，“演”不下去的吴某不得不摘下“面具”，法警将吴某带离拆除现场。“闹剧”收场的同时，“恶坝”也被拆除，农民心里的苦楚一并消散。
“怕难就不要干执行，执行就是要迎难而上”，郑林在执行中敢碰硬，不惧横。在他参与的案件中，被执行人有政府工作人员，有地痞“老赖”，面对“硬骨头”，郑林从不却步。
敢创新，敢担当，敢碰硬，郑林的执行艺术不断提高着办案数量和结案率。2016年以来，郑林结案1707件，追回标的额6.75亿元。郑林的执行艺术让他受到了黑龙江省高级人民法院的通报表扬，并多次被牡丹江市综治委员会评为“全市社会治理暨平安建设工作先进个人”。
