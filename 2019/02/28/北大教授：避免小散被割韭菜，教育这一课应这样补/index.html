北大教授：避免小散被割韭菜，教育这一课应这样补

发表于 2019-02-28 | 浏览量

股市难赚到钱，散户被割韭菜似乎已成既定命运？证监会主席易会满的最新表态，或许能让他们看到曙光。易会满27日在国新办举行的发布会上称，“将加强投资者教育，推动将投资者教育全面纳入国民教育体系。”

一财评论
股市难赚到钱，散户被割韭菜似乎已成既定命运？证监会主席易会满的最新表态，或许能让他们看到曙光。易会满27日在国新办举行的发布会上称，“将加强投资者教育，推动将投资者教育全面纳入国民教育体系。”
究竟如何才能将投资者教育落到实处？北京大学光华管理学院金融系陈佳、刘玉珍两位教授基于他们的研究建议，提高投资者教育是一个长期的过程，应将侧重点放在多元、多渠道、经验的教育上，而不只是短期的培训性质上。
陈佳、刘玉珍两位教授认为，加强投资者教育对于投资者做出更好的投资决策很有好处。有大量文献说明，更好的投资者教育，意味着投资者在投资中犯错误更少，投资效果更好更稳健。这些投资者的财富积累更有效，退休计划更好，更愿意投资股权市场，能选择交易费用更少的股权投资方式，能驾驭更高风险的投资产品。
加强投资者教育任重道远。从国际经验来看，发达国家的投资者教育在过去几十年中取得了长足的进步，但即使在这些国家中投资者教育不足仍然相当普遍，因此导致的投资决策错误也屡见不鲜。发展中国家的投资者教育水平相对普遍较低，给这些国家金融体系的健康发展带来了挑战。
国民教育体系对加强投资者教育至关重要。国际经验表明，对于同等智力和能力的人，更好的教育本身就意味着更好的对投资的理解。在此基础上，好的专业的投资者教育项目往往能显著增进投资者对投资的理解。推动投资者教育全面纳入国民教育体系是一项很好的举措。这项举措所面临的挑战是怎样找到效率-成本比例较高的措施。
对于不同的人群和地区，应视情况采取相应的教育措施。对教育基础薄弱的人群和地区，应针对其需要提高基本的投资者教育。对其他人群和地区，应结合投资市场实际情况进行与市场发展相对应的教育。
应花大力气培养一批熟练掌握投资基本知识和理念的教师。只有具有这些知识和理念的教师能高效高质提高投资者教育。
应将侧重点放在长期性基础性的教育上，而不是短期的培训性质的教育上。
，让投资者能根据这些理念做出对自己最有利的投资决策。
新建或者新兴的金融市场中投资者教育尤其重要。在这些较新的市场中，投资者教育不足所导致的投资错误更加普遍，从而带来投资损失，甚至引起恐慌，制约这些市场的健康发展。加强新建或者新兴金融市场的投资者教育需要相关各方一起努力，形成合力，推动这些市场健康发展。
，确保消费金融领域的相关营业机构向消费者提供足够透明的信息。与此同时，消费者应明确自己应负的责任。
对于复杂的产品或者不经常交易的产品，应适当限制个人投资者的投资。
，并对这些机构的投资者教育行为进一步加以规范。
