日前，珠三角九市2018年经济数据和2019年经济发展目标全部出炉。南方日报南方经济智库对这份成绩单进行梳理和分析后发现，各地新旧动能转换速度不减，创新驱动强劲，市场活力充沛。
新兴产业的发展速度，是判断发展动能转换成效的重要指标。《粤港澳大湾区发展规划纲要》也将“构建具有国际竞争力的现代产业体系”作为单独章节阐述。
新能源汽车、智能装备、新型显示、人工智能、生物医药、互联网，是广州重点培育的6个新支柱产业集群。2018年智能装备与机器人增长10.4％，新一代信息技术增长9.1％，轨道交通增长9％。此外，先进制造业和高技术制造业增加值占规模以上工业比重分别为59.7％和12.8％，表现稳健。
2018年，深圳第二产业产值达9962亿元，占GDP比重保持在40%以上。全年规模以上工业增加值增长9.5%，先进制造业增加值增长12%。
制造大市佛山，2018年规上装备业增加值增长6.4%，其中“工作母机”增长7.2%，109家企业入选广东制造业500强，增加51家。新兴产业的步伐也在加快，去年规模以上先进制造业增加值增长7.4%，占规模以上工业比重49%，提高2.4个百分点。
东莞2018年工业五大支柱产业实现增加值2706.73亿元，增长7.7%。倍增计划成效逐步显现，全市263家倍增计划工业企业实现增加值758.95亿元，增长8.8%。
惠州的电子信息和石化产业大项目不断落地而保持较快增长。中海油惠炼二期全面建成投产，大亚湾石化区炼化一体化规模跃居全国第一。新增年主营业务收入超百亿元工业企业3家，总数达10家。全市规上工业总产值增长8.3%，规上工业企业利税总额增长12%。
珠海工业发展引人瞩目。2018年，该市规模以上工业增加值达1082.14亿元、增长14%，工业对经济增长的贡献率达76.1%。鲲龙AG600飞机水上首飞成功，云洲智能中国首艘导弹无人艇成功试射等，成为其靓丽的产业名片。此外，珠海在引资引才方面成绩喜人，2018年新设外商投资企业3973家，同比增长153.9%。
不断优化的营商环境正在成为大湾区城市发展的源头活水。
广州新登记市场主体和企业分别增长25.45％和35.25％。新增上市公司10家、新三板挂牌企业21家。新兴消费快速发展，网上商店零售额增长15％，飞机融资租赁增长61％，邮轮进出港旅客增长19.3％。
深圳开展营商环境改革、工程建设项目审批“深圳90”改革、智慧城市和数字政府建设等走在全国前列，全年为企业减负超过1400亿元，民营经济增加值增长9.6%，全年新设立企业29万户、总量增长11.6%。
在佛山，民营工业对全市工业增长贡献率达到80.1%，2018年新登记市场主体15.72万户，增长30.9%，新增主营业务收入超100亿元企业2家、累计达20家。
东莞则着力推动中小微企业做大做强，全市新增“四上”企业3800家、“小升规”企业2784家，增量均为全省第一。此外，该市2018年规上工业企业突破1万家，排名全省第一。
具有全球影响力的国际科技创新中心是粤港澳大湾区的五个战略定位之一。《粤港澳大湾区发展规划纲要》明确提出，将建设全球科技创新高地和新兴产业重要策源地。
2018年，广州新增高新技术企业超过2000家，高新技术企业数量突破1万家，科技创新企业超过20万家，国家科技型中小企业备案入库数居全国城市第一。专利和发明专利授权量分别增长45％和20％，技术合同成交额增长1倍以上。展望未来，“广州—深圳—香港—澳门”科技创新走廊将为广州科创增添新动力，中新广州知识城、南沙庆盛科技创新产业基地等被明确为粤港澳大湾区的重大创新载体。
深圳国家级高新技术企业总量超过1.4万家，数量居全国第二，仅次于北京。全社会研发投入占GDP比重、PCT国际专利申请量全国领先，数字经济发展走在全国前列。
佛山获批建设国家创新型城市，预计研发经费支出占地区生产总值比重2.42%，新增国家高新技术企业1350家、累计达3900家。
2018年，中山高企认定数预计超过2300家，全市发明专利申请量和授权量实现快速增长，中国科学院药物创新研究院华南分部、哈工大无人装备与人工智能创新中心等科技创新平台的落地为中山未来的高质量发展蓄力。
江门创新发展迅速，2018年高新技术企业突破1200家，增长70%，增速连续两年排珠三角第一，规模以上工业企业研发机构覆盖率达56.53%。
今日的高质量投资，为明日的高质量发展打下基础，梳理各城市的产业布局和政策走向，未来的发展空间也就自然浮现。
从投资看，广州发展后劲雄厚。2018年工业投资增长53.8%，占全市固定资产投资总额的比重为16.0%，其中制造业投资增长65.6%，高技术制造业投资增长1.5倍。
2019年，广州将培育若干个千亿级产业新集群，建设省级示范基地、工业互联网标识解析国家顶级节点，提前布局量子通信、航空维修、轨道交通、天然气水合物开采及衍生技术等前沿产业。
《粤港澳大湾区发展规划纲要》提出，深化港深、澳珠合作，加快广佛同城化建设。广佛同城化有望迎来新一轮发展契机。
根据规划，佛山将与广州深化产业协同发展，聚焦先进装备制造、汽车、新一代信息技术、生物医药与健康等产业，共建若干个万亿级产业集群，并力争国家高新技术企业达5000家，规模以上工业企业研发机构建有率提高到55%。
作为粤港澳大湾区四大中心城市之一的深圳，将努力打造全球创新创业和投资发展最佳首选地，建设世界一流的光明科学城，加快推进材料基因组等大科学装置建设，确保基础研究投入比例不低于财政科技专项资金的30%，新增国家级高新技术企业2000家以上。
东莞提出，将重点布局新一代信息技术、高端装备制造、新材料、新能源、生物科技等新兴产业，加快“芯屏机核”等关键技术突破，力争先进制造业、高技术制造业增加值占比分别达到53%和39.5%。
惠州则将用8—10年时间打造石化能源新材料、电子信息两大万亿级产业集群，谋划建设具有国际水平的石化技术研发中心，大力发展超高清视频产业，把稔平半岛建设成为大湾区能源基地和科技创新中心。
中山提出，将大力发展健康医药、新一代信息技术、高端装备制造等战略性新兴产业，高起点建设先进制造业集群。
珠海将加快发展新一代信息技术、高端装备制造、生物医药、新能源及新能源汽车、节能环保、数字创意、新材料等产业，积极布局大数据、人工智能等产业，形成新一代信息技术产业集聚区。珠海横琴作为粤港澳大湾区重大合作平台之一，将建设国际休闲旅游岛，并将加快推进横琴澳门青年创业谷和粤澳合作产业园等重大合作项目建设，研究建设粤澳信息港。
肇庆则将推动制造业高质量发展和推进粤港澳大湾区产业共建作为两个主攻方向，力争全年引导280家工业企业实施技术改造，力争装备制造业增加值增长14%、投资增长17%。
